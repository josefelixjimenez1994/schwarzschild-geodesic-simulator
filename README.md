# Simulador de geodésicas en el espacio-tiempo de Schwarzschild  
# Schwarzschild Spacetime Geodesic Simulator

**Autor / Author:** José Félix Jiménez Justo

Herramienta de simulación numérica desarrollada en Python con interfaz gráfica en PyQt6 para el estudio de trayectorias geodésicas en el espacio-tiempo de Schwarzschild.

Numerical simulation tool developed in Python with a PyQt6 graphical interface for the study of geodesic trajectories in Schwarzschild spacetime.

## Descripción / Description

La herramienta permite simular geodésicas temporales, correspondientes a partículas masivas, y geodésicas nulas, correspondientes a fotones. El programa representa trayectorias orbitales, potenciales efectivos, radios característicos, puntos de retorno, procesos de captura, escape, dispersión y efectos de dilatación temporal.

The tool allows the simulation of timelike geodesics, corresponding to massive particles, and null geodesics, corresponding to photons. The program represents orbital trajectories, effective potentials, characteristic radii, turning points, capture, escape, scattering processes, and time dilation effects.

## Funcionalidades principales / Main features

* Integración numérica de geodésicas temporales y nulas.

* Representación de trayectorias orbitales.

* Visualización del potencial efectivo.

* Análisis de radios característicos: horizonte de sucesos, ISCO y esfera de fotones.

* Estudio de puntos de retorno, captura, escape y dispersión.

* Comparación temporal entre observador estático y partícula masiva simulada.

* Exportación de resultados y figuras.

* Numerical integration of timelike and null geodesics.

* Representation of orbital trajectories.

* Visualization of the effective potential.

* Analysis of characteristic radii: event horizon, ISCO, and photon sphere.

* Study of turning points, capture, escape, and scattering.

* Temporal comparison between a static observer and a simulated massive particle.

* Export of results and figures.
