Simulador de geodésicas en el espacio-tiempo de Schwarzschild

Autor: José Félix Jiménez Justo

La herramienta está desarrollada en Python con interfaz gráfica en PyQt6. Permite simular trayectorias geodésicas de partículas masivas y fotones en la métrica de Schwarzschild, representar el potencial efectivo, analizar radios característicos, puntos de retorno y efectos de dilatación temporal.

Requisitos:
- Python 3.10 o superior
- Librerías indicadas en requirements.txt

Instrucciones de ejecución:

1. Descomprimir la carpeta.
2. Abrir una terminal o PowerShell dentro de la carpeta.
3. Instalar las dependencias con:

   pip install -r requirements.txt

4. Ejecutar el programa con:

   python main.pyw

En Windows también puede probarse con:

   py main.pyw
